## Overview
**Shadow AI** is a digital twin career assistant platform for women. It helps women silently build careers while managing other responsibilities. The platform focuses on converting invisible labor into career capital using AI-powered tools.

Core Features:
- Personalized AI-driven career guidance
- Goal mapping and journey planning
- Micro-learning hub with progress tracking
- Resume builder and job/grant finder for women
- Secure authentication with Supabase
- Responsive, modern UI using React + TailwindCSS

---

## Tech Stack
- **Frontend:** React, TypeScript, TailwindCSS, Framer Motion
- **Backend:** Supabase (Authentication, Firestore, Storage)
- **AI Tools:** OpenAI/Cohere for AI assistance and prompts
- **Other:** Vite for build tooling, localStorage for session management
